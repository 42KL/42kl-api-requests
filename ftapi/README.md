# ftapi

A lightweight Python helper package for authenticating with the 42 API and fetching 42 intra data.

## Overview

`ftapi` provides:
- `FtApi`: OAuth2 backend authentication for the 42 API.
- `FtCursus`: helper functions for cursus-related API calls and date calculations.
- `FtUser`: dataclasses for building user and cursus-user payloads for 42 API requests.

This package is designed to simplify 42 API access from scripts inside the `ftapi/` workspace.

## Features

- OAuth2 client credentials authentication using `requests-oauthlib`.
- Pagination-aware GET requests with automatic page iteration.
- Environment-based credentials loading from `.env`.
- Helpers for cursus users, cursus projects, and date range computation.
- Simple user and cursus-user payload dataclasses with validation helpers.

## Installation

Install required dependencies from the repository root:

```bash
pip install -r requirements.txt
```

## Environment Setup

Create a `.env` file in the `ftapi/` directory with these values:

```env
42-UID=your_42_api_client_id
42-SECRET=your_42_api_client_secret
42-CAMPUS=your_campus_id
```

`FtApi` reads these values when instantiated.

## Usage

### Authenticate and fetch data

```python
from FtApi import FtApi

ft_api = FtApi()
print(ft_api)

users = ft_api.get(f"{ft_api.site}/v2/cursus/21/cursus_users?filter[campus_id]={ft_api.campus}")
print(len(users))
```

### Use cursus helpers

```python
from FtApi import FtApi
from FtCursus import get_cursus_users, get_cursus_projects

ft_api = FtApi()

cursus_users = get_cursus_users(ft_api, cursus_id="21", begin_date="2025-10-01")
projects = get_cursus_projects(ft_api, cursus_id="21")
```

### Create user payloads

```python
from FtUser import FtUser, FtCursusUser

user = FtUser(
    email="student@example.com",
    first_name="First",
    last_name="Last",
    kind="external",
    campus_id="1",
    pool_year="2025",
    pool_month="october"
)

if user.is_postable():
    payload = user.asdict()
    print(payload)
else:
    print(user.whats_wrong())

cursus_user = FtCursusUser(
    user_id="123",
    cursus_id="21",
    begin_at="2025-10-01T00:00:00.000Z"
)
print(cursus_user.asdict())
```

## Modules

### `FtApi`

- `load_env()`: Reads `42-UID`, `42-SECRET`, and `42-CAMPUS` from `.env`.
- `authenticate(uid, secret)`: Performs OAuth2 client credentials authentication.
- `get(url)`: Fetches JSON data with automatic pagination and a maximum page size of 100.

### `FtCursus`

- `get_cursus_users(ft_api, cursus_id, begin_date=None, end_date=None)`: Fetches cursus users filtered by campus and optional date range.
- `get_cursus_projects(ft_api, cursus_id)`: Fetches projects for a cursus.
- `compute_cursus_end_from_begin(cursus_id, begin_date)`: Computes cursus end date using built-in duration values.
- `compute_cursus_range_from_begin(cursus_id, begin_date)`: Computes both begin and end datetimes.

### `FtUser`

- `FtUser`: Dataclass for building 42 user payloads.
- `FtCursusUser`: Dataclass for building cursus-user payloads.
- `asdict()`: Returns non-None fields as a dictionary.
- `is_postable()`: Returns `True` when all required `FtUser` fields are present.
- `whats_wrong()`: Returns a message listing missing required fields.

## Notes

- This package assumes a local `.env` file and does not currently ship as a published PyPI package.
- The `get()` method pauses between paginated requests to help avoid API rate issues.
